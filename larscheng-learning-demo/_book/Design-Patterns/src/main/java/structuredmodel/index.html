结构型模式

结构型模型

描述如何将类或对象按照某种部剧组成更大的结构。

  • 类结构型模式：采用继承机制来组织接口和类
  • 对象结构型模式：采用组合或者聚合来组合对象

由于组合或者聚合关系比继承关系的耦合性更低，满足合成复用原则，所以对象结构型模式的灵活性更好
