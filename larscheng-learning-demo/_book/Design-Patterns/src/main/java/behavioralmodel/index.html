行为型模式

行为型模式用于描述程序在运行时复杂的流程控制

即描述多个类或对象之间如何协同工作共同完成单个对象无法完成的任务

其中涉及到了算法与对象职责的分配

分为两个大类

  • 类行为模式：继承机制
  • 对象行为模式：组合、聚合

组合聚合方式相比于继承机制耦合度更低，故对象行为模式灵活性更佳
